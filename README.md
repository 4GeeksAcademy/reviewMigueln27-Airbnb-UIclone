# Airbnb UI Clone

This project recreates an Airbnb-inspired interface with three routes:
- `/`
- `/catalog`
- `/rooms/[id]`

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- React 19

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm start
```
